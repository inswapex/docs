MetaBase Trading System

MetaBase is a decentralized token liquidity exchange protocol that supports real-time lossless exchange of platform tokens such as ERC20 and HRC20.

Market Making Mechanism

MetaBase's market-making mechanism is comprehensively improved on AMM, PMM and other mechanisms, and innovatively created IMM (Intelligent Market Maker) intelligent market-making mechanism, which supports AMM, PMM and other composite market-making, minimizing UNISWAP's impermanent losses and funds Exposure risk and improve capital utilization.

In order to avoid the friction between SWAP and the oracle, IMM adopts a step-by-step approach. In the early development of MetaBase, AMM was the mainstay and gradually transitioned to IMM.

MetaBase Market

MetaBase Market provides SWAP-related data aggregation and transparent analysis to help users more easily observe data and select participating targets.

Open Transaction Mining

MetaBase supports transaction mining for any transaction pair. Any anonymous user can freely choose to provide transaction mining funds for a transaction pair without any third-party review and docking.

When a user participates in a transaction with a transaction mining fund pool, he will receive a transaction mining reward based on the transaction fee.

In addition, in order to better improve user transaction activity and encourage users to effectively SWAP, INSWAP will start platform currency trading and mining activities in installments at appropriate times. The platform determines the mining time of each transaction and the number of rewards, and automatically distributes platform coins to users with a corresponding proportion of the transaction volume through the on-chain contract. The reward mechanism adopts the improved proof-of-stake mechanism (APOS) to automatically calculate the number of user rewards. The greater the user's transaction volume, the more rewards they will receive.

Multi-Dimensional Oracle System

The MetaBase oracle system, as an independent operating system, supports multi-dimensional composite market data and will provide users with a smooth and average market price reference.
