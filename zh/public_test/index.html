MetaBase社区公测说明

MetaBase是新一代去中心化的产业通证流通协议平台,历经长达8个月的自主研发,已初步完成平台建设和完善。 为鼓励用户参与,验证平台经济体系和压力能力,现向社区用户公开开展公测。

MetaBase采用开放式运营模式,任何有流量、有社群的用户都可以通过“升级LP->创建节点”的途径,自行成为社区节点,无需任何第三方审核和对接。

公测基本形式

MetaBase 本次公测以社区节点为主导,由KOL、社区长、流量群主等创立节点,所有参与用户必须加入所在节点,才能参加本次公测。

公测信息

公测时间: 2021-09-23 9:00 至 2021-09-30 16:00 (为期一周)

公测地址: https://heco256.metabase.cc/

开放公测节点： 最多5个

公测奖励地址: 最多2000个地址

公测内容: Swap、IDO、Staking、POS、Grant五个主要功能及相关模块

MetaBase公测基本规则

MetaBase公测遵循以下规则:

  1. 以社区节点为各个公测基本单元,由节点负责人组织、反馈相关测试活动。社区节点应完成用户的组织、公测通知的宣发、社群基础疑问解答等工作。
  2. 所有参与公测的用户必须加入所在节点,才能参与公测。
  3. 公测用户可在用户中心页面的公测展板区域,查看自身的公测进度,完成所有公测任务才能获得公测奖励(请注意,未完成任务的将不予发放公测奖励,敬请谅解)。
  4. 公测到期结束后,两周内发放用户和节点的平台币奖励。具体发放办法,请及时关注Twitter和Discord公告。

公测用户奖励规则

MetaBase公测用户总奖励数量为100000 META,最大奖励2000个用户地址,先到先得,超出部分不予发放。 公测用户至少完成以下任务:

  1. Swap次数>3
  2. IDO次数>1
  3. Staking次数>1
  4. POS次数>1
  5. 领取空投次数>2
  6. 加入节点

完成任务后,单个用户地址奖励 50 META (预计私募价在0.5 USDT,也可能调整)

公测节点奖励规则

MetaBase节点具有组织本社区节点参与公测、公测公告通知、社区基础疑问解答等基本工作。

节点奖励规则如下:

1.社区参与人数在100人以上,即可获得基础奖励 20000 META.
2.每推广一个用户地址,奖励20META,在基础奖励上累积。
3.节点最大全部奖励数量为40000 META.

公测节点需提前完成"创建节点"的工作,并将节点ID通知给社区用户。

节点创建教程:

https://metabaseex.github.io/docs/zh/tutorial/node#%E5%88%9B%E5%BB%BA%E8%8A%82%E7%82%B9

社区公测奖惩规则

  1. 经技术认定,机器人刷单和作恶者、作恶节点将不予发放META奖励。
  2. 测试期间平台故障导致的数据不准确，可提供交易Hash给节点。
  3. 所有奖励代币无需锁仓，在公测结束两周内批量发放。发放形式另行通知。
  4. MetaBase公测解释权归DAO社区所有，任何破坏公测的个人或节点将永久列入社区黑名单.
