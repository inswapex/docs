MetaBase IDO使用概述

MetaBase IDO平台是去中心化、无需第三方干预的产业通证众筹发行系统。

任何LP用户均可发起IDO众筹,全程无审核、无托管,并对众筹过程和结果负责。

发起IDO众筹需支付一定的平台币费用,以避免垃圾数据和洪水攻击。

  1. MetaBase IDO为无中心化审核机制,参与用户请谨慎参与IDO,请充分了解并加入项目社区后酌情考虑。
  2. IDO池创建适合项目方发起,不适合普通用户发起。
  3. IDO用户参与IDO时,应注意项目发起人和自身的锁仓情况。完全不锁仓的项目,属于高风险,请谨慎参与。
  4. IDO众筹的代币,至少在保护期后才能开始解锁,以避免恶意欺骗等情况,保护参与者的利益。

参与IDO

在IDO列表中,选择您想参与的IDO池,进入IDO详情页。

在IDO详情页,请仔细查看项目币种、地址(避免冒充)、项目锁仓策略等信息

当IDO时间达到时,用户点击"参与",输入IDO数量提交,就能参与IDO。

  1. IDO参与权限分为公开、白名单、邀请制、社区节点等四种,请确认您是否具备参与权限。白名单适合在私募阶段,面向特定对象(机构、天使投资人等);邀请制指当前账户的邀请人必须为IDO池创建人;社区节点指在指定的社区节点发起众筹,只有该节点的用户方能参与。
  2. MetaBase IDO为点对点、无审核机制,用户需自行承担项目真实性和价格风险。
  3. IDO用户参与后,在IDO结束时,即获得众筹代币(用户锁仓的IDO,需逐步解锁)。
  4. 项目方可选择解锁或不解锁IDO获得的目标代币,在平台予以公示。不接受锁仓策略和价格的用户,可以选择不参与。
  5. 为保护用户利益,MetaBase采取延迟交付机制,项目方至少在保护期之后,才能提取目标代币,即便没有创建锁仓策略。
  6. IDO池一旦创建,众筹数量、代币等即不能改变。如果项目方在创建池后,数据不正确,可关闭IDO池,但之前创建IDO池的费用将不予退回。

创建IDO池

只有LP用户才能创建IDO众筹池。
LP用户登录后,在IDO模块首页,点击"创建IDO池"。
填写IDO池信息和价格,规则,即可创建IDO池。 提交后,信息不可更改。 创建IDO池后,IDO池处于未激活状态。用户先充值IDO的众筹代币(按照创建时填写的众筹总量),就能激活IDO池。大约几分钟后,IDO池会显示在IDO列表中。

我的IDO池

在IDO首页,点击"我的IDO池",可查看"我参与的IDO" 和 "我创建的IDO"。

在"我参与的IDO"页面,可查看参与过的IDO项目,提取或解锁IDO代币。 在"我创建的IDO"页面,可查看创建过的IDO项目,进行IDO池管理。

创建IDO池后,建议先设置LP功能,然后再充值众筹代币。设置LP必须在激活前完成
