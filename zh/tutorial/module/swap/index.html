Inswap Swap使用概述#

Inswap Swap是一种创新的极速代币交换协议,它采用合约层交易方式进行代币交换,因此交易方式与传统DEX大不相同。

Swap交易#

Inswap平台中进行代币交换,是一种非常轻松、简单的事情。

进入Swap页面,在交易区域的左上角,选择交易币种,输入交易数量,选择"买入"或"卖出"即可。

Swap资产在合约内自动完成转账,无需进行授权等操作,交易手续费低,操作简单。

Swap注意事项
  1. Inswap合约内需先充值相应的资产,否则无法交易。
  2. 当交易对不存在时,Inswap会优先提供自动路由功能,间接完成目标交易。
  3. 当无法Route时,可能无法交易,需调整交易对或添加流动性。
  4. Swap最终获得金额以实际交易完成情况为准,页面数据仅供参考。

流动性管理#

任意用户都可以成为Inswap LP做市商,为交易对添加流动性,以获取IMM做市收益。
做市商有可能在平台的市场推广活动中,获得相关参与权利和优惠。 做市操作非常简单,在Swap交易区域,点击"添加流动性"或"移除流动性"即可。 做市商在移除流动性时,所获得的增量收益,即为做市收益和手续费分红。

交易挖矿#

为鼓励用户参与项目交易,平台还为项目方提供了新的激励工具--"交易挖矿"。 任何用户均可为交易对提供交易挖矿资金,交易挖矿资金只能为当前交易对的币种,不允许为其他币种。 点击"交易挖矿池"->"添加交易奖励",弹出交易挖矿信息页面。

交易奖励资金可重复添加,但一经添加,将无法退回,请添加前慎重操作
