MetaBase Staking使用概述

MetaBase Staking是一种去中心化、无中心化审核的质押挖矿系统。

任何LP用户均可发起Staking池,全程无审核、无托管。

发起Staking池需支付一定的平台币费用,以避免垃圾数据和洪水攻击。

  1. MetaBase Staking为无中心化审核机制,平台合理的规则对项目方和用户均无风险。
  2. Staking池创建适合项目方发起,不适合普通用户发起。
  3. Staking池属于固定期限、固定收益的保本挖矿,用户到期即可解锁全部本金和收益。
  4. Staking池创建后,在质押到期前,项目方无权动用池内资金。多余的奖励代币,将在质押结束后,自动结算退回。

参与Staking

在Staking列表中,选择您想参与的Staking池,点击"质押"按钮,填写质押数量,提交至链上,即完成本次质押。

Staking为固定期限质押,在质押到期前,用户和项目方均无法干预。到期后,用户可解锁本金和收益

  1. Staking参与权限分为公开、白名单、邀请制、社区节点等四种,请确认您是否具备参与权限。白名单适合针对大客户,面向特定对象(机构、天使投资人等);邀请制指当前账户的邀请人必须为Staking池创建人;社区节点指在指定的社区节点发起Staking,只有该节点的用户方能参与。
  2. MetaBase Staking为点对点、无审核机制,用户和项目方均无风险。
  3. Staking只是质押挖矿的工具,并不对二级市场价格负责,用户参与前须知这一点。

创建Staking池

只有LP用户才能创建Staking池。
LP用户登录后,在Staking模块首页,点击"创建质押池"。
填写Staking池信息和奖励比例,即可创建Staking池。 提交后,信息不可更改。 创建Staking池后,Staking池处于未激活状态,不会显示在公开列表。用户先充值Staking的奖励代币(按照创建时填写的质押总量、奖励比例、周期核算),就能激活Staking池。大约几分钟后,Staking池会显示在Staking列表中。

我的Staking池

在Staking首页,点击"我的质押",可查看"我参与的质押池" 和 "我创建的质押池"。

在"我参与的质押池"页面,可查看参与过的Staking项目,解锁质押资金、领取质押分红。 在"我创建的质押池"页面,可查看创建过的Staking项目,进行Staking池管理。

创建Staking池后,建议先设置LP功能,然后再充值奖励代币。

设置LP必须在激活前完成
