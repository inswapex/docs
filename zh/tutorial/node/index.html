MetaBase节点概述

为鼓励社区自治,MetaBase遵循"以贡献奖励收益"的自治原则,实行DAO节点制度。
DAO节点制度是一种开放式运营思维，所有用户均可通过销毁平台币成为LP用户,然后成为DAO。

社区节点 : 即DAO社区流量贡献者,社区节点可邀请项目和用户入驻MetaBase平台。

节点收益 : DAO节点参与平台所有相关收益的分成,包括但不限于Swap手续费、上币费、IDO分成、Staking收益管理费、POS管理费等。

  1. DAO节点一般是社区KOL、团队长、市场运营人员等,具备一定市场推广能力和流量,在MetaBase平台实行流量变现。
  2. DAO节点名称需规范名称,不得使用非法名称(政治、反动、黄色、侮辱性等),否则MetaBase可能封禁节点,并且不予解封和发放收益。
  3. DAO节点享有终身管道收益,不受任何第三方审核,由智能合约自动、实时结算。
  4. 节点可通过自主运营、组织社区活动(定向IDO、Staking、POS、空投等),提高节点吸引力和影响力。
  5. 所有节点均为匿名节点,建设私域流量、用户和社区。MetaBase暂不节点做任何排名和公示。
  6. 任何用户只要知晓节点ID,即可加入节点,无需审核。

创建节点

在用户中心页面的个人区域,点击右上角导航菜单"DAO管理"->"我的DAO",

在节点页面,选择"我创建的DAO"。
如果已经创建DAO节点,将显示节点管理页面,不显示"创建DAO"按钮。
单击"创建节点"按钮,按照页面提示,填写节点名称,通过Metamask提交到链上,即可轻松创建DAO节点。

创建节点可能需要一定的META费用。节点名称切勿使用非法名称(反动、政治、色情、侮辱性等),否则可能被封禁,造成损失

DAO管理

社区节点可轻松管理自己的节点,查看节点用户地址和节点统计数据。

加入DAO

任何用户知晓节点ID,即可加入DAO节点。

加入节点只有一次机会,确认后将无法更改。

加入节点后,可参与节点活动,享受节点福利(空投等)。 加入节点后,查看我加入的节点信息。
