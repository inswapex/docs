MetaBase项目生态

MetaBase创始初衷,便是面向项目方友好的开放式DAO运营生态,这在当今的区块链行业,属于非常罕见和可贵的。

MetaBase为广大项目方提供了自由、公平、公正、无审核、无托管的运营平台。

基本原则

1、项目上币无需审核、自主上币。
任何项目但需支付一定的平台币作为平台费用,以筛选优质资产,减少无效长尾币种的干扰。

2、上币即拥有MetaBase的全部生态工具,自由使用任何模块。

3、项目方在MetaBase发起IDO众筹、Staking质押挖矿、POS活期挖矿、社区投放等均无需审核,在支付一定的平台币费用后,将自动上线。全程无第三方干预和有条件审批。

项目上币

项目上币由项目社区任意用户提交,向合约发起链上认证,自动完成。

在MetaBase App主页的左侧导航栏,点击"Token认证",

在弹出的信息框里,输入代币地址(必填)和推荐节点ID(0x开头，选填),
点击"确定提交"。 然后在Metamask里提交上链即可。

项目上币规则
  1. 项目上币无需第三方审批,无需审批。上币费用自动从当前登录账户余额扣除。
  2. 上币后即可使用平台的全部功能模块。
  3. 恶意代币、欺诈性代币或违反政治、历史等社会公序的代币名称,一经发现,可能会被列入代币黑名单。上币账户可能会被封禁,以维护社区秩序。
  4. 代币合约可能存在一定的安全风险,请自行甄选鉴别。MetaBase无法判定,无法确认相关责任。
