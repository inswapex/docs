How to add network to Metamask wallet#

The Inswap platform supports Metamask wallet. To use the Inswap platform, you need to prepare a Metamask wallet, whether you are from the PC or Web. If there is no public chain network deployed by Inswap in your current Metamask, please refer to this document to configure it to correctly access the Inswap platform.

If you have configured Metamask, please ignore this document. If your environment is not mentioned in this article, please understand, but you can refer to similar methods

Metamask plugin installation#

Metamask wallet supports Chrome kernel browsers, such as Chrome (recommended), Firefox, Brave, Edge, etc. Download from Metamask official website: https://metamask.io/download.html

Add Metamask network#

You can add a network to Metamask in the following two ways, choose one of them.

1. Add network via CoinList#

Browser open https://chainlist.org/, Enter Heco in the search bar (similar to other networks), Click "Connect" in the search results, and the next step is just fine.

2. Manually add network#

In the Metamask wallet, manually enter the network parameters and add the target network. In the browser plug-in bar, open the MetaMask wallet and click on the network list.

Pull the list to the bottom and select "Custom RPC".

Enter the corresponding network parameters and save.
